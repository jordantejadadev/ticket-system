# Ticket System

## Requisitos

- Java 17
- Node.js
- PostgreSQL
- Docker (más adelante)

## Configuración

Copiar:

```bash
.env.example
```

como:

```bash
.env
```

y completar las credenciales.

## Backend

```bash
cd backend
./mvnw spring-boot:run
```

## Frontend

```bash
cd frontend
npm install
npm run dev
```
